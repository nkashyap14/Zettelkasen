# Top-Down (Agglomerative) Clustering sklearn

```
from sklearn.cluster import AgglomerativeClustering

agg = AgglomerativeClustering(n_clusters=3)

agg.fit(data)

print('{}\n'.format(repr(agg.labels_)))
```

- Does not use [[centroids]]
- No predict function
