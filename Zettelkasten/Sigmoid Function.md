# Sigmoid Function

$$
e
$$
- A mathematical [[function]] that produces an S shaped curve and maps real values to a range between 0 and 1
- Essentially a useful function to squash output values between a probabilistic range
$$
f(x) = \frac{1}{1 + e^{-x}}
$$
