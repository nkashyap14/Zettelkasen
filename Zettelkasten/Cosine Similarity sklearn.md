# Cosine Similarity sklearn

```
from sklearn.metrics.pairwise import cosine_similarity
data = np.array([
  [ 1.1,  0.3],
  [ 2.1,  0.6],
  [-1.1, -0.4],
  [ 0. , -3.2]])
  
cos_sims = sklearn.metrics.pairwise.cosine_similarity(data)
print('{}\n'.format(repr(cos_sims)))
```
