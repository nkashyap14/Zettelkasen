- Maps to 3 layers of the [[OSI Model]]
	- [[Session Layer]]
	- [[Presentation Layer]]
	- [[Application Layer]]
- Encompasses various high level [[protocol|protocols:]]
	- [[SMTP]]
	- [[FTP]]
	- [[DNS]]
	- [[HTTP]]
- Provides interfaces and protocols for apps to communicate with the network
