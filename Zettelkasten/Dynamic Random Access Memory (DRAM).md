# Dynamic Random Access Memory (DRAM)

## Definition:

- [[memory]] built as an [[Integrated Circuit]]; it provides random access to any location. 
- Access times are 50 [[nanoseconds]] and cost per gigabyte in 2020 was 3-6 dollars
