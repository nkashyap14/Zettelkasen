# Discrete Uniform Distributed Random Variable

- Takes one out of a range of contiguous integer values with equal probability
- PMF:
	- 0 Otherwise
$$
p_x(k) = \frac{1}{b - a + 1} if k = a,a+1,...,b
$$

- Var(X)
